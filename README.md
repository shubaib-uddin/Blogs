# ✍️ Agentic AI Blog Writing System

An advanced multi-agent AI blogging system built with LangGraph, Ollama, and Streamlit that automatically researches, plans, and generates complete technical blog posts
with a modern UI dashboard.

## 🚀 Features

* 🤖 Multi-Agent LangGraph Workflow
* 🔍 Automatic Web Research with Tavily
* 🧠 Intelligent Research Routing
* 📋 Structured Blog Planning
* ⚡ Parallel Section Generation
* 🎨 Modern Streamlit Dashboard
* 📚 Previous Blog History
* 📄 Markdown Export Support
* 🔗 Automatic Citations & Sources
* 🧩 Modular Agent Architecture

## 🧠 Workflow

1. User enters a blog topic
2. Router agent decides research strategy
3. Tavily gathers relevant web evidence
4. Planner agent creates blog structure
5. Worker agents generate sections in parallel
6. Reducer combines everything into a final blog
7. Blog is saved automatically as Markdown

## 🛠️ Tech Stack

* Python
* LangGraph
* LangChain
* Ollama
* Tavily Search
* Streamlit
* Pydantic
